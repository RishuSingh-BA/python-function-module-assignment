# Functions & Modules in Python

## Task 1: Factorial Function
- Defines a function `factorial(n)` that calculates the factorial using recursion.
- Calls the function with a sample number (e.g., 5).
- Displays the result.

## Task 2: Math Module Calculations
- Asks the user for a number.
- Uses Python's `math` module to calculate:
  - Square root
  - Natural logarithm (base e)
  - Sine (in radians)
- Displays the results.

### How to Run
1. Clone this repository.
2. Run `Task1_Assig_3.py` for Task 1.
3. Run `Task2_Assig_3.py` for Task 2.
